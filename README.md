# Clipboard Manager – User Documentation

## User Guide (EN)

### Overview
Clipboard Manager is a simple tool that automatically stores your copied text and allows you to quickly access previous clipboard entries. The application works in the background and can be opened anytime using a global shortcut.

### Features
- Automatic clipboard history
- Up to 30 stored clipboard items
- Global shortcut: Ctrl + Shift + V
- Manual copy back to clipboard
- Export and import history (JSON)
- Automatic hiding to system tray

### Getting Started
After launching the application, it will minimize. To display the application, use the keyboard shortcut mentioned above.

### Global Hotkey
Ctrl + Shift + V – Opens or hides the main window.

### Main Window Layout
1. **Clipboard History List** – Shows time and short preview, double-click to copy.
2. **Buttons:**
   - **Copy:** Copy selected item back to clipboard.
   - **Clear All:** Deletes all stored items.
   - **Export:** Save history to JSON.
   - **Import:** Load history from JSON.

### Automatic Clipboard Tracking
Automatically stores new copied text if valid and non-duplicate.

### Exiting
Open the window and click X. Tray icon disappears – app fully closed.

## Uživatelský manuál (CZ)

### Přehled
Clipboard Manager je nástroj, který automaticky ukládá text, který kopírujete, a umožňuje rychlý přístup k předchozím položkám schránky. Aplikace běží na pozadí a lze ji otevřít globální zkratkou.

### Funkce
- Automatická historie schránky
- Až 30 uložených položek
- Globální zkratka: Ctrl + Shift + V
- Kopírování zpět do schránky
- Export a import historie (JSON)
- Automatické skrytí po spuštění

### Začínáme
Po spuštění aplikace se minimalizuje. Pro zobrazení aplikace použijte již zmíněnou klávesovou zkratku.

### Globální klávesová zkratka
Ctrl + Shift + V – otevře nebo skryje hlavní okno.

### Hlavní okno
1. **Seznam historie** – zobrazuje čas a náhled, dvojklik zkopíruje.
2. **Tlačítka:**
   - **Kopírovat:** Zkopíruje vybranou položku.
   - **Vymazat vše:** Smaže historii.
   - **Export:** Uloží do JSON.
   - **Import:** Načte historii z JSON.

### Automatické sledování schránky
Uloží nový text, pokud je platný a neopakovaný.

### Ukončení
Otevřete okno a klikněte na X. Ikona zmizí – aplikace ukončena.
